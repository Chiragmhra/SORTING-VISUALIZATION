1. Bubble Sort

Bubble sort compares adjacent elements and swaps them if they are in the wrong order. After each pass, the largest element "bubbles up" to its correct position.

2. Insertion Sort

Insertion sort builds the sorted list one element at a time by picking the current element and inserting it into its correct position in the sorted part of the array.

3. Selection Sort

Selection sort repeatedly finds the minimum element from the unsorted part and places it at the beginning of the array.
